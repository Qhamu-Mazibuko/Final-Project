# Income Survey Web Application

## Overview
This project is a Flask-based web application designed to collect user income and expense data. The data is stored in MongoDB, exported to CSV, and analyzed using Python.

## Features
- Web form for user input
- MongoDB data storage
- CSV export
- Data visualization using Matplotlib

## Technologies Used
- Python (Flask, Pandas, Matplotlib)
- MongoDB
- HTML

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run MongoDB locally

3. Start Flask app:
   python app.py

4. Export data:
   python export_to_csv.py

5. Open Jupyter Notebook:
   jupyter notebook

## Deployment
The application can be hosted on AWS EC2 by installing dependencies and running the Flask server.
